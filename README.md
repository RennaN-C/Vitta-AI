# 🚀 Vitta AI - Trading Terminal

Sistema inteligente de análise de ações desenvolvido com foco em performance e design premium. O projeto integra o poder do **Python (FastAPI)** no backend com a agilidade do **React (Vite)** no frontend, utilizando o banco de dados **Neon (PostgreSQL)** para persistência de dados.

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS, Lucide Icons.
* **Backend:** Python 3.10+, FastAPI, Uvicorn.
* **Banco de Dados:** Neon (Serverless PostgreSQL).
* **IA/LLM:** Groq API (Llama 3).
* **Estilização:** Dark Mode Extreme com acentos Gold/Neon.

## 🚀 Funcionalidades
- [x] Autenticação completa via Neon.
- [x] Perfil de usuário customizado (Pormade Portas).
- [x] Dashboard interativo com interface Glassmorphism.
- [x] ChatBot integrado para análise de mercado (Em desenvolvimento).
